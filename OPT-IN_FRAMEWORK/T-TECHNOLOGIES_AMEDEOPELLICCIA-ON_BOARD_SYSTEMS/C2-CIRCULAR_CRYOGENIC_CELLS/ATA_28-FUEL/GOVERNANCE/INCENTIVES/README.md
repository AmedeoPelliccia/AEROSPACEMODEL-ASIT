# ATA 28 – Token Incentive Programme

**System:** AMPEL360 Q100 LH₂ Fuel System (C2 Circular Cryogenic Cells)
**ATA Chapter:** 28 – Fuel
**Version:** 1.0.0

## Overview

This directory manages the token-based incentive programme for ATA 28
contributors. Tokens are awarded for verified contributions that advance
system development milestones.

## Contents

| File                    | Purpose                              |
|-------------------------|--------------------------------------|
| `TT_ALLOCATION.yaml`   | Token pools by activity type         |
| `TT_DISTRIBUTION.csv`  | Ledger of awarded tokens             |
| `TT_POLICY.md`         | Earning, vesting, and redemption rules |
