## Compliance Methods

| Code | Method | Description |
|------|--------|-------------|
| MC1 | Analysis | Engineering analysis and calculation |
| MC2 | Test | Ground or flight test |
| MC3 | Similarity | Comparison with certified design |
| MC4 | Inspection | Physical inspection |
| MC5 | Equipment qualification | DO-160 qualification |

## Application to SC-28-H2-001

Primary means: MC1 (Analysis) + MC2 (Test) for pressure,
thermal, and leak requirements.

## Application to SC-28-CRYO-002

Primary means: MC2 (Test) + MC5 (Equipment qualification)
for cryogenic temperature operation.
