# Quality Assurance Plan — 28-10-00

## Inspection Points

| Item | Method | Acceptance Criteria |
|------|--------|-------------------|
| Inner shell welds | X-ray / UT | Per AWS D17.1 |
| MLI layer count | Visual | 40 layers ± 0 |
| Vacuum level | Gauge | < 1E-3 mbar |
| Leak test | He leak | < 1E-9 mbar·L/s |

and cryogenic compatibility verification.
