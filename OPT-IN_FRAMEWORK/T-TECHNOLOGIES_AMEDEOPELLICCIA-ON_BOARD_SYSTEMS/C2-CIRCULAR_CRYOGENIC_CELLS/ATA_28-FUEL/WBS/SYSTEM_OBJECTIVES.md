# ATA 28 – System Objectives

**System:** AMPEL360 Q100 LH₂ Fuel System (C2 Circular Cryogenic Cells)
**ATA Chapter:** 28 – Fuel
**Version:** 1.0.0

## Measurable Objectives

### Performance

| ID      | Objective                        | Target        | Verification |
|---------|----------------------------------|---------------|--------------|
| OBJ-01  | Daily boil-off rate              | < 0.5%/day    | Test         |
| OBJ-02  | Leak detection response time     | < 3 s         | Test         |
| OBJ-03  | Tank capacity                    | 1 200 kg LH₂  | Analysis     |
| OBJ-04  | Operating temperature            | −253 °C       | Test         |
| OBJ-05  | Maximum operating pressure       | 5.0 bar       | Test         |

### Programme

| ID      | Objective                        | Target        | Verification |
|---------|----------------------------------|---------------|--------------|
| OBJ-06  | Certification completion         | Q4 2027       | Review       |
| OBJ-07  | Requirement traceability         | 100%          | Inspection   |
| OBJ-08  | Unresolved safety KNOTs at CDR   | 0             | Review       |

### Safety

| ID      | Objective                        | Target              | Verification |
|---------|----------------------------------|----------------------|--------------|
| OBJ-09  | H₂ leak probability (major)     | < 1E-9 per FH       | Analysis     |
| OBJ-10  | Pressure relief reliability      | < 1E-7 per FH       | Analysis     |
| OBJ-11  | Fire/explosion prevention        | < 1E-9 per FH       | Analysis     |

## Traceability

Each objective traces to one or more WBS work packages and is verified at
the designated lifecycle gate (PDR, CDR, TRR, or FRR).
