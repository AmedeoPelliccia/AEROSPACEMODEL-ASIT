# ATA 28 – System Vision

**System:** AMPEL360 Q100 LH₂ Fuel System (C2 Circular Cryogenic Cells)
**ATA Chapter:** 28 – Fuel
**Version:** 1.0.0

## Target End-State

A fully certified LH₂ fuel system meeting EASA CS-25 requirements with
Special Conditions SC-28-H2-001 (Hydrogen Storage and Distribution) and
SC-28-CRYO-002 (Cryogenic Temperature Handling).

## Key Performance Targets

| Parameter               | Target          |
|-------------------------|-----------------|
| Daily boil-off rate     | < 0.5%          |
| Tank capacity           | 1 200 kg LH₂   |
| Operating temperature   | −253 °C         |
| Max operating pressure  | 5.0 bar         |
| Requirement traceability| 100%            |

## Vision Attributes

- Full lifecycle traceability from requirements through certification.
- Operational readiness for entry into service.
- S1000D-compliant technical documentation baseline.
- BREX-governed content with deterministic agent support.
